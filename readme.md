# Node/Mysql Boilerplate
&nbsp;

## Start project: steps
1. clone repo (git clone https://github.com/fabiochiquezi/node-boilerplate.git)
2. create a .env file (touch .env)
3. install project (yarn install)
4. run project: (yarn docker || yarn dev2)
